Reports

Tools useful when working with SAP Reports in Kyano Validate

WIP Report Tools

  1. WIP - Hierarchical Parser generator - generates arguments for Hierarchical Parser Profile based on report header (Text defining layout)
  2. WIP - Hierarchical Parser check - You execute only BI + compare for the report in Validate, copy the output and paste it in this checker and it checks if your profile is set up correctly.
  3. WIP - Hierarchical Parser visualizator - You insert Hierarchical profile + data from Image (Spool), and the tool displays your data as a table
