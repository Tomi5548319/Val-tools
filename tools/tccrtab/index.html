TC Cross-reference Tables

Extract table names from cross-reference messages.

How to access cross-reference in TC:

  1. Go to "Global" > "Selections" / "Mappings"
  2. Select a row (Rule)
  3. Click on "Cross-reference"
  4. Copy all rows (Including number, status and message fieds)
  5. Paste the data here to get a list of tables which are using the selection / mapping
